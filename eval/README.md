# 统计学 RAG 评测目录

这个目录用于存放当前统计学 RAG 项目的评测集、评测脚本、指标说明、judge 选型说明，以及各轮评测输出结果。

## 建议阅读顺序

如果是第一次接触这个项目，建议按下面顺序阅读：

1. `指标体系与调优导读.md`
   先理解 RAG 评测到底在评什么、每个指标是什么意思、应该怎样看调优结果。

2. `EVAL_METRICS.md`
   这是第一版正式指标定义稿，说明哪些指标是主指标、哪些是辅助指标。

3. `JUDGE_MODEL_SELECTION.md`
   说明为什么最终选择 `zai-org/GLM-4.6` 作为 judge model。

4. `评测与优化记录.md`
   这里记录了基线版本、三轮优化过程、各轮指标变化，以及为什么保留当前最终版本。

5. `runs/20260404_192700/summary.json`
   这是当前最终保留版本的完整评测摘要。

## 文件说明

- `eval_set_template.jsonl`
  第一版 36 条单轮评测集。

- `run_eval.py`
  自动化评测脚本。负责运行检索、生成、judge 评分，并汇总指标。

- `EVAL_METRICS.md`
  第一版正式指标定义文档。

- `JUDGE_MODEL_SELECTION.md`
  judge model 选型文档。

- `指标体系与调优导读.md`
  面向项目组成员的入门导读，适合先学指标和调优思路，再去看实验结果。

- `评测与优化记录.md`
  记录各轮优化目标、改动、结果与结论。

- `runs/`
  各次评测的输出结果目录，包含：
  - `summary.json`：结构化汇总结果
  - `summary.md`：便于快速阅读的摘要
  - `per_sample_results.jsonl`：逐题详细结果
